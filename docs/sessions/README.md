# Session Archive

This directory stores archived copies of previous `docs/SESSION-CONTINUATION.md` files.

## Purpose

Archived session docs preserve historical handoff state without cluttering the active continuation file.

## Rules

- keep the active handoff in `docs/SESSION-CONTINUATION.md`
- move completed session snapshots here
- keep filenames date-based for easy scanning

## Naming Convention

- `SESSION-YYYY-MM-DD.md`
