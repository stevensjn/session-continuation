# Roadmap

This roadmap is intentionally lightweight in form, but on the `large-project` branch it now illustrates how the same repository can support a much broader product workflow.

## Phase Overview

| Phase | Status | Goal |
|---|---|---|
| Phase 1 | ✅ | Establish documentation-first repository structure |
| Phase 2 | ✅ | Define the minimal React Hello World behavior and routing docs |
| Phase 3 | 🔄 | Add the large-project branch workflow with brief plus tracker plus DDL separation |
